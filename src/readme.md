## General info

## Requirements
* Node v14.16.1

## Running
Dependencies should be installed before runnin the application

To run application use:
```
npm run start
```
Application is working on port 3000 by default

## Testing
To run test use
```
npm run test
```
or 
```
npm run test:watch
```
